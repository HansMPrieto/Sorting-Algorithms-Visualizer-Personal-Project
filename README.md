# Sorting-Algorithms-Visualizer-Personal-Project

Sorting Algorithms Visualizer that implements the following sorting algorithms:

- Mergesort
- Quicksort
- Heapsort
- Selection Sort
- Bubblesort
- Insertion Sort
- Odd-Even Sort
- Cocktail Sort
- Insertion Sort
